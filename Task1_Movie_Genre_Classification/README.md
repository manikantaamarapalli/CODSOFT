# Movie Genre Classification

## Objective
Predict movie genres using movie plot summaries and textual descriptions.

## Dataset is provided by the @CODSOFT FOR TRAIN,TEST AND EVALUATE Model.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression
- Naive Bayes
- Support Vector Machine (SVM)

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 58.36% |
| Naive Bayes | 52% |
| SVM | 57% |

## Best Model
Logistic Regression achieved the highest accuracy of 58.36%.
