# Movie Genre Classification

## Objective
Predict movie genres using plot summaries and textual information.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- TF-IDF
- Logistic Regression
- Naive Bayes
- SVM

## Models Compared

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 57.94% |
| Naive Bayes | 52% |
| SVM | 57% |

## Best Model
Logistic Regression achieved the highest accuracy of 58.36%.
